# Simple Static Website

A simple website structure using:
- normalize (or reset)
- browser-sync
- jshint
- gulp
- gulp-autoprefixer
- gulp-concat
- gulp-jade
- gulp-jshint
- gulp-load-plugins
- gulp-plumber
- gulp-util
- gulp-sass
- gulp-sourcemaps
- gulp-uglify
- gulp-yuicompressor
- main-bower-files
